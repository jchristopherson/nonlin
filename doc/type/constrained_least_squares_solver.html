type, public, extends(constrained_equation_solver) :: constrained_least_squares_solver

Defines a constrained least-squares solver using Powell's trust region method. The method seeks a correction $p$ that reduces the residual vector $r(x)$ while remaining inside a trust region of size $\Delta$ and the bounds $x^L \le x \le x^U$. The trial step is formed from a Gauss-Newton step a steepest-descent step and a dogleg combination with $t$ chosen so that $|p| = \Delta$. If the trust-region step is rejected, a backtracking line search is used to improve the step acceptance.
