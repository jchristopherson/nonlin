type, public, extends(line_search_optimizer) ::  bfgs

Defines a Broyden–Fletcher–Goldfarb–Shanno (BFGS) optimizer for minimization of smooth functions of multiple variables. The method uses a quasi-Newton inverse Hessian approximation and the update with $s_k = x_{k+1}-x_k$ and $y_k = \nabla f(x_{k+1})-\nabla f(x_k)$.

type, public, extends(equation_optimizer) ::  line_search_optimizer

A base class for optimizers that improve a trial step with a line search. The search direction $p_k$ is combined with a step length $\alpha_k$ through and the accepted step satisfies a sufficient decrease condition.

type, public, extends(equation_optimizer) ::  nelder_mead

Defines a direct-search optimizer based on Nelder and Mead's simplex method. The algorithm maintains a simplex of $n+1$ vertices and updates it by reflection, expansion, contraction, and shrinking steps to reduce the objective value $f(x)$.
